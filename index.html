AWS Glossary — Explained Like I’m 5

Snowball

A physical device AWS ships you to transfer massive amounts of data.

Think of it as a big moving truck for your digital files.

IAM Role

A temporary permission set that lets AWS services talk to each other.

Like giving a worker a visitor’s badge so they can use specific tools for a while.

S3 Bucket

A named storage container in Amazon S3 for your files.

It’s like a big labeled folder where you toss in documents, photos, or backups.

Multi-AZ

Running your app across multiple Availability Zones for safety and uptime.

Like having the same store in different neighborhoods so business keeps running if one closes.

Read Replica

A duplicate database made for reading data faster.

Like having extra cashiers in a store to serve more customers quickly.

NAT Gateway

Allows private servers to safely connect to the internet.

Think of it as a secure mailbox that lets you send and receive letters without revealing your home address.

Placement Group

A way to organize EC2 instances for performance or availability.

It’s like seating friends together so they can chat faster.

Cost Explorer

A tool to visualize and track your AWS spending.

Like checking a piggy bank report to see where your money went.

Well-Architected Framework

A set of best practices to build secure and efficient systems on AWS.

Like a rulebook for building a strong, safe, and fun treehouse.

VPC (Virtual Private Cloud)

A private section of AWS where you can define your own network.

Imagine a fenced playground where only your friends can come in and play.

IAM (Identity and Access Management)

A service that controls who can do what in your AWS account.

Like a security guard who checks IDs before letting people into different rooms.

RDS (Relational Database Service)

A managed service to run relational databases easily.

Like a neat filing cabinet that organizes all your data so you can find it quickly.

Lambda

A service to run code without managing servers.

Like a robot that runs errands instantly whenever you call it, without needing a full shop.

CloudWatch

Monitors AWS resources and applications in real time.

Like a babysitter making sure everything in your house is working fine.

CloudFormation

A tool for defining AWS resources with templates.

Like instructions for building a Lego castle — AWS follows the plan automatically.

Elastic Load Balancer (ELB)

It distributes traffic across multiple servers.

Like a traffic cop directing cars into different lanes so no one lane gets jammed.

Auto Scaling

It automatically adjusts the number of servers depending on demand.

Like hiring more helpers when your shop gets busy and letting them go when it’s quiet.

Route 53

A DNS service for managing domains and routing traffic.

It’s AWS’s GPS, helping people find your site on the internet.
